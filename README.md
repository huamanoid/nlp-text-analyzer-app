### NLP app with Streamlit & Spacy
A Simple NLP app for text analysis with drag and drop for files.


#### Requirement
+ pip install
	- streamlit
	- pandas
	- matplotlib
	- seaborn
	- spacy
		python -m spacy download en
	- textblob
	- neattext
	- wordcloud

+ File Processing
	- docx2txt
	- pyPDF2
	- pdfplumber
